ooccor - Object Oriented C Code Operations in Ruby

Swiss army knive commend line tool (and a collection of Ruby scripts) to operate on C-ish source code.

Currently only C is supported, support for C++ planned.

Support for Java, Objective-C, C# might also be added one day ...
